Average – Quantitative Aptitude

Introduction

Average is a fundamental concept in mathematics used to represent the central value of a data set. It is widely asked in competitive exams like SSC, Banking, CAT, GATE, Railway and Placement tests.

Average Formula

Average = Sum of Observations / Number of Observations
For n numbers x₁, x₂, x₃ ... xₙ
Average = (x₁ + x₂ + ... + xₙ) / n

Important Average Formulas

First n Natural Numbers
Sum = n(n+1)/2
Average = (n+1)/2
Squares of first n Natural Numbers
Sum = n(n+1)(2n+1)/6
Average = (n+1)(2n+1)/6
Cubes of first n Natural Numbers
Sum = [n(n+1)/2]²
Average = n(n+1)² / 4
First n Odd Numbers
Sum = n²
Average = n
First n Even Numbers
Sum = n(n+1)
Average = n+1

Important Tricks & Concepts

  • If one value increases by k, average increases by k/n
  • If average increases by k, total sum increases by nk
  • Combined Average = (Sum1 + Sum2) / (n1 + n2)
  • Equal Distance Average Speed = 2xy / (x + y)
  • Weighted Average = (w₁x₁ + w₂x₂ + ...)/Total Weight

Solved Examples

Q1: Average of 12 numbers is 68. First 7 avg = 65, next 4 avg = 72. Find 12th number.
Answer: 73
Q2: Speed 80 km/hr going, 50 km/hr returning. Average speed?
Answer: 61.54 km/hr
Q3: Distances: 250, 200, 180, 300. Average?
Answer: 232.5
Q4: Weighted scores 80 (40%), 90 (60%).
Answer: 86%
Q5: Weekly temperatures average?
Answer: 26.14°C
