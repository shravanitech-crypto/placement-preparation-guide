Accenture Placement Preparation Guide

This module provides detailed information about Accenture company recruitment, eligibility criteria, placement syllabus, job roles, salary and preparation strategy for students preparing for campus placements.

About Accenture Company

Company Overview

  • Accenture is a global professional services company.
  • Founded in 1989.
  • Headquarters located in Dublin, Ireland.
  • Operates in more than 120 countries.
  • Over 700,000 employees worldwide.
  • Provides IT services, consulting and outsourcing.
  • Works with many Fortune 500 companies.

Major Services

  • Technology Consulting
  • Cloud Computing Solutions
  • Artificial Intelligence
  • Cyber Security Services
  • Digital Transformation
  • Business Process Outsourcing
  • Application Development and Maintenance

Eligibility Criteria

Academic Qualification

  • B.E / B.Tech (All branches)
  • MCA / M.Sc (Computer Science / IT)
  • BCA / BSc IT for some roles
  • Minimum 60% marks in academics
  • No active backlogs
  • Good communication skills

Required Skills

  • Programming knowledge (Java / Python / C++)
  • Basic Data Structures
  • Database concepts and SQL
  • Logical reasoning ability
  • Problem solving skills
  • Good teamwork and communication

Accenture Recruitment Process

Selection Process

  • Online Aptitude Test
  • Cognitive Ability Assessment
  • Technical Assessment
  • Coding Test
  • Technical Interview
  • HR Interview

Online Test Sections

  • Quantitative Aptitude
  • Logical Reasoning
  • Verbal Ability
  • Coding Problems
  • Email Writing Test

Job Roles at Accenture

Associate Software Engineer

  • Develop software applications
  • Write and test code
  • Fix software bugs
  • Work with development teams

Application Developer

  • Design software applications
  • Implement business solutions
  • Work on cloud technologies
  • Ensure system performance

Data Analyst

  • Analyze business data
  • Create reports and dashboards
  • Work with SQL and Excel
  • Use Python for analysis

Technical Support Engineer

  • Provide technical support
  • Solve system issues
  • Maintain software systems
  • Assist customers
