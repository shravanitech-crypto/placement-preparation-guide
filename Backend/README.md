# Placement Preparation Guide

Placement Preparation Guide is a comprehensive web platform developed to assist students in preparing for campus placements. The system provides aptitude learning modules, practice tests, coding preparation resources, interview questions, and AI-powered guidance to improve placement readiness.

## Features
- User Registration and Login
- Aptitude Topics and Learning Material
- Theory with Examples
- Practice Questions and Mock Tests
- Coding Preparation Resources
- Interview Preparation
- Progress Tracking
- AI-Powered Practice Assistance
- Responsive and User-Friendly Interface

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Hosting: Vercel (Frontend), Render (Backend)

## Objective
To provide a centralized platform where students can prepare for aptitude, coding, and interviews efficiently and improve their chances of placement success.
